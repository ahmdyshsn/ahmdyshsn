A simple and practical accounting app for managing customers, debts, payments, and balances.

**Tech Stack:** Flutter, Django REST API, MySQL

- Debt and payment tracking
- Total due and credit balance
- Secure login
- Arabic-first interface

A scholarship and education management mobile app built with Flutter.

**Tech Stack:** Flutter, GetX, Dio, REST API

- Arabic RTL interface
- Dark and light mode support
